# Data

**Dataset:** World Values Survey Wave 7  
**Question:** Q272 — "What language do you normally speak at home?"  
**Coverage:** 66 countries  

Download from: https://www.worldvaluessurvey.org/WVSDocumentationWV7.jsp  
File needed: `WVS_Cross-National_Wave_7_csv_v6_0.csv`  
Place it in this folder before running the scripts.
